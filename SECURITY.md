# Security Policy

## Supported Versions
This repository provides a reference implementation of the JERT Network components.
Only the `main` branch is considered for security-related discussions.

## Reporting a Vulnerability
If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it responsibly by contacting:

Please include:
- a clear description of the issue,
- steps to reproduce (if applicable),
- potential impact.

We will acknowledge receipt and respond as soon as possible.

## Scope
This policy applies to:
- smart contracts,
- API gateway code,
- corporate and mobile wallet applications.

Production deployment parameters and validator governance are handled off-chain and
are **out of scope** for this repository.
