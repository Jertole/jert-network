# JERT Smart Contracts — USD Edition
Contains all Solidity logic:
- JERTToken.sol — ERC‑20, fixed supply 1 trillion, USD‑denominated valuation standard.
- TreasuryMultisig.sol — 2/3 governance multisig.
- KYCRegistry.sol — on‑chain whitelist.
- ComplianceGateway.sol — sender/receiver filtering.
- LeaseContract.sol — prepaid service leasing (USD‑based).

All contracts use NatSpec for AIFC alignment.
